Connect to Firebase

This tracker stores all invoices in Firebase Firestore so your data is available from any browser or device. Paste your Firebase project config below to get started.

  1. Go to console.firebase.google.com and open your project (or create a new one).
  2. Click the gear icon → Project settings
  3. Under Your apps, add a Web app if you haven't (or use existing).
  4. Copy the firebaseConfig object and paste it below.
  5. In Firestore → Rules, set read/write to true (for private use) or add auth rules.

Config is saved in this browser's localStorage so you only need to enter it once per device.
